### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。

### 三、功能描述
系统分为（身份）：普通用户、员工、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 订单信息管理
6. 物流运输管理


**员工功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 商品信息管理
5. 入库信息管理
6. 出库信息管理
7. 订单信息管理
8. 物流运输管理



**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 用户管理
4. 员工管理
5. 库位信息管理
6. 商品分类管理
7. 商品信息管理
8. 入库信息管理
9. 出库信息管理
10. 供应商管理
11. 订单信息管理
12. 物流运输管理
